De bloques a C++ — en un vistazo

Comprende cómo se traduce tu lógica a un sketch real. Perfecto para dar el salto a la programación en texto.

BloquesVista previa
[ Al iniciar ]
  configurar pin 9 como SALIDA
[ Repetir por siempre ]
  leer potenciómetro A0 → valor
  brillo = map(valor, 0..1023 → 0..255)
  analogWrite(9, brillo)
C++ generado.ino
void setup(){
  pinMode(9, OUTPUT);
}

void loop(){
  int valor = analogRead(A0);
  int brillo = map(valor, 0, 1023, 0, 255);
  analogWrite(9, brillo);
}

Qué incluye el MVP

Componentes iniciales
  • LED, resistencia, botón, potenciómetro
  • Servo, buzzer
  • Sensores simples (p. ej., DHT simulado)
Placas y energía
  • Arduino UNO (base), ESP32 (mapeo inicial)
  • Fuentes 3.3V/5V/9V/12V y GND (simulados)
Programación
  • Bloques básicos: IO, control, variables, map
  • Vista C++ y exportación .ino
Limitaciones del MVP
  • No emula el MCU real; simulación visual equivalente
  • Guardado local (LocalStorage) en la primera versión

Para docentes y aulas

Pensado para clases introductorias de electrónica y programación: objetivos claros, baja fricción y resultados visibles.
